## 推送方法

### 方法1：使用 GitHub Desktop（推荐，最简单）

1. **下载安装 GitHub Desktop**
   - 访问：https://desktop.github.com/
   - 下载并安装

2. **添加本地仓库**
   - 打开 GitHub Desktop
   - File -> Add Local Repository
   - 选择目录：`E:\Huawei\ai_transform`
   - 点击 Add

3. **发布到 GitHub**
   - 点击 "Publish repository" 按钮
   - 选择仓库名：`AI_Transform`
   - 选择账户
   - 点击 "Publish repository"

### 方法2：使用 Personal Access Token

#### 步骤1：生成 Token

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 填写信息：
   - Note: `AI Transform Push Token`
   - Expiration: 选择有效期（建议90天或更长）
   - 勾选权限：**repo**（全部）
4. 点击 "Generate token"
5. **立即复制 Token**（只显示一次）

#### 步骤2：推送代码

在 PowerShell 或 Git Bash 中执行：

```bash
cd E:\Huawei\ai_transform
git push -u origin main
```

当提示输入时：
- **Username**: 输入您的 GitHub 用户名
- **Password**: 输入刚才复制的 Token（不是密码）

### 方法3：使用 SSH（需要配置）

#### 步骤1：生成 SSH 密钥

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
# 按 Enter 使用默认路径
# 可以设置密码或直接按 Enter
```

#### 步骤2：复制公钥

```bash
cat ~/.ssh/id_ed25519.pub
# 复制输出的内容
```

#### 步骤3：添加到 GitHub

1. 访问：https://github.com/settings/keys
2. 点击 "New SSH key"
3. Title: 填写描述（如：My Computer）
4. Key: 粘贴刚才复制的公钥
5. 点击 "Add SSH key"

#### 步骤4：修改远程地址并推送

```bash
cd E:\Huawei\ai_transform
git remote set-url origin git@github.com:LZY18829039855/AI_Transform.git
git push -u origin main
```

## 常见问题

### Q: 提示 "Authentication failed"
A: 
- 确保使用 Personal Access Token 而不是密码
- 检查 Token 是否有 `repo` 权限
- Token 可能已过期，重新生成

### Q: 提示 "Permission denied"
A:
- 检查是否有仓库的写入权限
- 确认仓库地址正确

### Q: 提示 "Repository not found"
A:
- 确认仓库地址正确
- 确认有访问权限
